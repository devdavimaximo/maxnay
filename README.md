# 🐾 Maxnay Frontend

Este é o código-fonte da aplicação front-end da **Maxnay**, uma landing page premium para clínicas veterinárias.

---

### 🚀 Visualização Online
Acesse o projeto em: **[https://clinicamaxnay.vercel.app](https://clinicamaxnay.vercel.app)**

---

### 📂 Sobre este Repositório
Este projeto foi criado para compor meu **portfólio**, demonstrando domínio em:
- **Next.js 15 (App Router)**
- **React 19**
- **Tailwind CSS 4**
- **Framer Motion**

Para a documentação completa, instruções de instalação e detalhes técnicos, por favor, consulte o **[README principal na raiz do projeto](../README.md)**.

---

### 🛠️ Scripts Disponíveis

Na pasta `frontend`, você pode rodar:

- `npm run dev`: Inicia o servidor de desenvolvimento.
- `npm run build`: Cria a versão de produção.
- `npm start`: Inicia a versão de produção.
- `npm run lint`: Executa o linter para verificar o código.
